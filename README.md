# Portfolio Website

## Node.js setup
Install dependencies and start the development server:

```powershell
npm install
npm run dev
```

Create a production build:

```powershell
npm run build
npm run preview
```

## Optional Python virtual environment (Windows PowerShell)
If you want to install Python dependencies from `requirements.txt`:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```
